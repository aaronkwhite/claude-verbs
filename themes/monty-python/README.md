# Monty Python Spinner Verbs

Absurdist British humor for your Claude Code spinner. 39 verbs spanning Holy Grail, Life of Brian, and Flying Circus.

## Sample Verbs

- **Ni-ing** - Knights Who Say Ni
- **Running Away** - "Run away! Run away!"
- **Expecting the Spanish Inquisition** - Nobody expects it
- **Pining for the Fjords** - Dead Parrot sketch
- **Consulting the Book of Armaments** - Holy Hand Grenade

## Files

- [monty-python.json](./monty-python.json) - Configuration file
- [monty-python.md](./monty-python.md) - Full verb list with references

## Installation

Copy `monty-python.json` to your Claude Code settings directory, or add the `spinnerVerbs` configuration to your existing settings.
