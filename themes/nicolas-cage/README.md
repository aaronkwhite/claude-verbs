# Nicolas Cage Theme

Custom spinner verbs for Claude Code inspired by the legendary Nicolas Cage.

## Sample Verbs

- **Stealing the Declaration of Independence** for 30s
- **NOT THE BEES-ing** for 30s
- **Putting the Bunny Back in the Box** for 30s
- **Taking His Face... Off** for 30s
- **Going Full Cage** for 30s
- **Forging a Chainsaw Battle Axe** for 30s

## Files

- [`nicolas-cage.json`](./nicolas-cage.json) - Configuration file (40 verbs)
- [`nicolas-cage.md`](./nicolas-cage.md) - Full verb list with movie references

## Installation

Add to your Claude Code settings:

```json
{
  "spinnerVerbs": {
    "mode": "replace",
    "verbs": [...]
  }
}
```

See `nicolas-cage.json` for the complete verb list.
