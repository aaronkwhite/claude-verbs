# Bob Ross Theme

A calm, wholesome Claude Code spinner verb theme inspired by Bob Ross and *The Joy of Painting*.

## Sample Verbs

- Beating the Devil Out of It for 30s
- Adding Happy Little Trees for 30s
- Making Happy Accidents for 30s
- Finding Friends for Trees for 30s
- Building Happy Clouds for 30s

## Files

- [bob-ross.json](bob-ross.json) - JSON configuration for Claude Code
- [bob-ross.md](bob-ross.md) - Full verb list with references

## Usage

Copy `bob-ross.json` to your Claude Code configuration, or use the Quick Copy block in `bob-ross.md`.
