# Parks and Recreation Theme

Spinner verbs for Claude Code inspired by the Parks and Recreation Department of Pawnee, Indiana.

## Sample Verbs

- **Treating Myself** - Tom and Donna's annual tradition
- **Eating All the Bacon and Eggs** - Ron Swanson's legendary order
- **Literally the Best-ing** - Chris Traeger's enthusiasm
- **Achieving Peak Duke Silver** - Ron's secret jazz persona
- **Being Network Connectivity Problems** - Andy's FBI alias

## Files

- [parks-and-rec.json](./parks-and-rec.json) - Configuration file (40 verbs)
- [parks-and-rec.md](./parks-and-rec.md) - Full reference with verb explanations

## Usage

Copy `parks-and-rec.json` to your Claude Code settings directory, or merge the `spinnerVerbs` configuration into your existing settings.
