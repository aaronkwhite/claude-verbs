# Trump Mockery Theme

A **satirical roast** of Donald Trump's verbal tics, lies, ego, and incompetence turned into Claude Code spinner verbs.

## Sample Verbs

```
Covfefe-ing for 30s
Many People Are Saying-ing for 30s
Staring at the Eclipse for 30s
Injecting Disinfectant Theoretically for 30s
Person Woman Man Camera TV-ing for 30s
Sharpie-ing the Weather Map for 30s
Tremendous-ing Bigly for 30s
```

## Files

| File | Description |
|------|-------------|
| [trump-mockery.json](./trump-mockery.json) | JSON config for Claude Code (43 verbs) |
| [trump-mockery.md](./trump-mockery.md) | Full documentation with reference table |

## Installation

Copy the contents of `trump-mockery.json` to your Claude Code settings file:

```bash
# macOS
~/.claude/settings.json

# Or project-level
.claude/settings.json
```

## Disclaimer

This is complete satire and mockery. Every verb references something absurd that actually happened or was actually said. If you're offended, you might be taking it too seriously. Or not seriously enough. Believe me.
