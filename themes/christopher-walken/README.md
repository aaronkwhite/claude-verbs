# Christopher Walken Spinner Verbs

Transform your Claude Code spinner into something with a little more... intensity.

This theme captures the essence of Christopher Walken's unforgettable presence - the dramatic pauses, the dancing, the staring, and of course, the cowbell.

## Sample Verbs

```
Needing More Cowbell for 30s
Pausing... Dramatically for 30s
Hiding Watches in Uncomfortable Places for 30s
Weapon of Choosing for 30s
Dancing in Hotel Hallways for 30s
```

## Files

- **[christopher-walken.json](./christopher-walken.json)** - Ready-to-use JSON configuration (40 verbs)
- **[christopher-walken.md](./christopher-walken.md)** - Full verb list with movie/sketch references

## Installation

Copy the JSON config to your Claude Code settings:

```bash
cat christopher-walken.json
```

Or reference the full documentation in `christopher-walken.md` for context on each verb.
