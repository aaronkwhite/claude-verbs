# Snoop Dogg & Dr. Dre Theme

West Coast G-funk spinner verbs for Claude Code, celebrating the legendary Snoop Dogg and Dr. Dre partnership.

## Sample Verbs

- Dropping It Like It's Hot for 30s
- Sippin' on Gin and Juice for 30s
- Still D.R.E.-ing for 30s
- Fo Shizzling My Nizzle for 30s
- Hitting the Next Episode for 30s

## Files

- [snoop-and-dre.json](./snoop-and-dre.json) - JSON config for Claude Code
- [snoop-and-dre.md](./snoop-and-dre.md) - Full verb list with references

## Usage

Copy `snoop-and-dre.json` to your Claude Code settings or use the Quick Copy JSON block in the markdown file.
