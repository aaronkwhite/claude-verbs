# Seinfeld Spinner Verbs

A Seinfeld-themed verb pack for Claude Code spinner messages. Transform your CLI experience into a show about nothing.

## Sample Verbs

- **Yada Yada-ing** for 30s
- **Experiencing Shrinkage** for 30s
- **No Soup For You-ing** for 30s
- **Being Master of Domain** for 30s
- **Serenity Now-ing** for 30s

## Files

- [seinfeld.json](./seinfeld.json) - Configuration file for Claude Code
- [seinfeld.md](./seinfeld.md) - Full verb list with references and episode citations

## Installation

Copy `seinfeld.json` to your Claude Code configuration directory or merge the `spinnerVerbs` block into your existing settings.

## Stats

- **Total Verbs:** 40
- **Mode:** Replace (replaces default verbs entirely)
- **Mix:** Famous catchphrases + deep-cut episode references
