# Dad Jokes Theme

Peak dad energy for your Claude Code spinner. Every loading message brings the warmth of a freshly mowed lawn, the wisdom of thermostat management, and the comfort of a couch nap you definitely weren't taking.

## Sample Verbs

```
Adjusting the Thermostat for 30s
Hi Hungry I'm Dad-ing for 30s
Finding Deals at Home Depot for 30s
Just Resting My Eyes for 30s
Defending My Cargo Shorts for 30s
```

## Files

- **[dad-jokes.json](dad-jokes.json)** - Drop-in config (42 verbs)
- **[dad-jokes.md](dad-jokes.md)** - Full verb list with references

## Installation

Copy `dad-jokes.json` to your Claude Code settings or merge the `spinnerVerbs` config into your existing setup.
