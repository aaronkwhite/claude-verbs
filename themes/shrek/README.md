# Shrek Theme

Ogre-powered spinner verbs for Claude Code, featuring 40 verbs from all four Shrek films.

## Sample Verbs

- Getting Out of My Swamp
- Checking Layers
- Making Waffles
- Interrogating Gingerbread Men
- Doing the Roar

## Files

- **[shrek.json](./shrek.json)** - Configuration file for Claude Code
- **[shrek.md](./shrek.md)** - Full verb list with references and quick copy JSON

## Usage

Copy `shrek.json` to your Claude Code configuration directory, or use the JSON from `shrek.md` directly in your settings.
