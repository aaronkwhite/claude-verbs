# Arrested Development Spinner Theme

Custom spinner verbs for Claude Code inspired by the Bluth family's misadventures.

## Sample Verbs

- Making a Huge Mistake for 30s
- Blue-ing Myself for 30s
- Doing Chicken Impressions for 30s
- Committing Light Treason for 30s
- Popping a Forget-Me-Now for 30s

## Files

- [arrested-development.json](./arrested-development.json) - Theme configuration (40 verbs)
- [arrested-development.md](./arrested-development.md) - Full reference with quotes and explanations

## Installation

```bash
cat arrested-development.json >> ~/.claude/settings.json
```

Or manually copy the `spinnerVerbs` section into your Claude Code settings.
