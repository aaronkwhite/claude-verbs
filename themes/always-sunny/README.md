# It's Always Sunny in Philadelphia Theme

A Claude Code spinner verb theme inspired by It's Always Sunny in Philadelphia. Transform your coding sessions with the chaotic energy of the Gang's schemes, Dennis's delusions, Charlie's illiteracy, and Frank's questionable life advice.

## Sample Verbs

- **Implicating** - "Because of the implication"
- **Demonstrating Value** - The D.E.N.N.I.S. System
- **Offering Eggs in Trying Times** - Frank's comfort strategy
- **Becoming a Golden God** - Dennis's peak delusion
- **Wildcarding** - "Wildcard, bitches! Yeehaw!"

## Files

- [`always-sunny.json`](./always-sunny.json) - Configuration file for Claude Code
- [`always-sunny.md`](./always-sunny.md) - Full reference with all 41 verbs and their origins

## Usage

Copy `always-sunny.json` to your Claude Code configuration directory, or merge the `spinnerVerbs` configuration into your existing settings.

## What It Looks Like

```
Implicating for 30s
Cultivating Mass for 12s
Bird Law-ing for 45s
Peaking Spectacularly for 8s
```
