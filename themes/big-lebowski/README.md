# Big Lebowski Theme

Custom spinner verbs for Claude Code inspired by *The Big Lebowski* (1998).

## Sample Verbs

- **Abiding** - The Dude's core philosophy
- **Marking It Zero** - Walter's bowling rule enforcement
- **Entering a World of Pain** - Walter's iconic threat
- **Rolling on Shabbos** - Walter's religious observance
- **Tying the Room Together** - The rug's essential purpose

## Files

- [`big-lebowski.json`](./big-lebowski.json) - Configuration file for Claude Code
- [`big-lebowski.md`](./big-lebowski.md) - Full verb list with references

## Usage

Copy `big-lebowski.json` to your Claude Code configuration directory, or merge the `spinnerVerbs` configuration into your existing settings.

The verbs are designed to read naturally with duration suffixes (e.g., "Abiding for 30s").
