# Jeff Goldblum Theme

A Claude Code spinner verb theme capturing the chaotic charm, signature stammering, and philosophical musings of Jeff Goldblum across his most iconic roles.

## Sample Verbs

```
Life Uh Finding a Way for 30s
Must Go Faster-ing for 30s
Being Afraid, Very Afraid for 30s
Chaos Theory-ing for 30s
Uploading the Virus-ing for 30s
Grandmaster-ing for 30s
```

## Files

- [jeff-goldblum.json](./jeff-goldblum.json) - Configuration file for Claude Code
- [jeff-goldblum.md](./jeff-goldblum.md) - Full verb list with references and quick copy JSON

## Usage

Copy `jeff-goldblum.json` to your Claude Code configuration directory or merge the `spinnerVerbs` section into your existing settings.
