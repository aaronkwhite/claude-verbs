# Noir Detective Theme

Hardboiled spinner verbs for Claude Code, straight from the rain-soaked streets of 1940s-50s film noir.

## Sample Verbs

- Following the Dame
- Pouring Another Whiskey
- Staring Through Venetian Blinds
- Trailing a Shadowy Figure
- Confronting the Femme Fatale

## Files

- [noir-detective.json](./noir-detective.json) - Claude Code configuration file (41 verbs)
- [noir-detective.md](./noir-detective.md) - Full verb list with references and quick copy block

## Usage

Copy `noir-detective.json` to your Claude Code settings directory, or grab the JSON block from the markdown file.
