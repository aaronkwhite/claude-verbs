# Arnold Schwarzenegger Spinner Verbs

Action hero spinner verbs for Claude Code, featuring 40 verbs inspired by Arnold Schwarzenegger's most memorable movie moments.

## Sample Verbs

- Getting to the Chopper for 30s
- Being Back for 30s
- Terminating Processes for 30s
- Letting Off Some Steam for 30s
- Crushing Enemies for 30s

## Files

- [arnold-schwarzenegger.json](./arnold-schwarzenegger.json) - JSON configuration for Claude Code
- [arnold-schwarzenegger.md](./arnold-schwarzenegger.md) - Full reference with all verbs and movie sources

## Usage

Copy `arnold-schwarzenegger.json` to your Claude Code configuration directory or copy the JSON block from the markdown file directly into your settings.
