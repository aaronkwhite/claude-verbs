# The Office Theme for Claude Code Spinner Verbs

> "I'm not superstitious, but I am a little stitious." - Michael Scott

A collection of 40 spinner verbs inspired by NBC's The Office (US). These verbs transform iconic moments, quotes, and running gags from Dunder Mifflin into action format that reads naturally with timing appended (e.g., "Declaring Bankruptcy for 30s").

## Verb Reference Table

| Verb | Reference |
|------|-----------|
| Declaring Bankruptcy | Michael's misunderstanding of bankruptcy law - "I DECLARE BANKRUPTCY!" |
| That's What She Said-ing | Michael's signature inappropriate catchphrase |
| Bearing Beets | Jim impersonating Dwight - "Bears. Beets. Battlestar Galactica." |
| Calling a Meeting in the Conference Room | Michael's constant unnecessary meetings |
| Waiting for Pretzel Day | Stanley's one true joy - "I wake up every morning in a bed that's too small" |
| Parkour-ing Through the Office | Michael, Dwight, and Andy's parkour cold open |
| Scheduling a Meeting That Could Have Been an Email | Universal office frustration, Dunder Mifflin style |
| Putting Stuff in Jello | Jim's classic prank on Dwight's stapler |
| Ignoring Toby | Michael's inexplicable hatred - "If I had two bullets..." |
| Running a Dundies Ceremony | The annual Dunder Mifflin awards show |
| Threatening to Burn the Building Down | Milton... wait, wrong office. But also Kevin's energy |
| Looking at the Camera Meaningfully | Jim's signature fourth-wall-breaking reaction |
| Microwaving Fish in the Break Room | Kevin's ongoing culinary crimes |
| Identity Theft-ing | "IDENTITY THEFT IS NOT A JOKE, JIM!" |
| Studying at Cornell | Andy's constant Ivy League references - "I went to Cornell. Ever heard of it?" |
| Forming an Alliance | Jim and Dwight's shifting workplace alliances |
| Crunching Numbers | Kevin doing "keleven" math |
| Starting a Fire Drill | Dwight's safety demonstration gone wrong |
| Scissor Me-ing | Michael and Oscar's inappropriate "scissoring" moment |

| Making Chili | Kevin's famous (and tragic) chili spill |
| Questioning if This is Real Life | The documentary crew's perspective |

| Talking Head-ing | The documentary interview format |
| Sipping Wine Out of a Coffee Mug | Meredith's workday coping mechanism |
| Turning It Off and On Again | Nick the IT guy's solution (before he left for Philly) |
| Attending Sensitivity Training | Post-Michael-incident mandatory training |
