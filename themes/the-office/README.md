# The Office Theme

Spinner verbs inspired by NBC's The Office (US) for Claude Code.

## Sample Verbs

- Declaring Bankruptcy
- That's What She Said-ing
- Putting Stuff in Jello
- Starting a Fire Drill
- Making Chili

## Files

- [the-office.json](./the-office.json) - JSON configuration (40 verbs)
- [the-office.md](./the-office.md) - Full reference with quotes and explanations

## Usage

Copy the contents of `the-office.json` into your Claude Code configuration to replace the default spinner verbs with The Office themed alternatives.
