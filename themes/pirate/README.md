# Pirate Theme

Arr, matey! Transform your Claude Code spinner into a swashbuckling adventure on the high seas.

## Sample Verbs

- "Swabbing the Deck for 30s"
- "Searching for the Black Pearl for 30s"
- "Avoiding the Kraken for 30s"

## Files

- [pirate.json](pirate.json) - Configuration file (40 verbs)
- [pirate.md](pirate.md) - Full verb list with references

## Installation

Copy `pirate.json` to your Claude Code settings directory or merge the `spinnerVerbs` configuration into your existing settings.
