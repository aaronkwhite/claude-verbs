# Mob Classics Theme

> "As far back as I can remember, I always wanted to be a gangster."
> â€” Henry Hill, Goodfellas

A collection of spinner verbs inspired by the greatest mob stories ever told: **Goodfellas** and **The Sopranos**. These verbs capture the essence of wiseguy life, from making collections to handling waste management.

## Verbs

| Verb | Reference |
|------|-----------|
| Getting the Papers | Goodfellas - Jimmy's repeated line about getting documentation |
| Forgetting About It | Classic Italian-American phrase, "Fuggedaboutit" |
| Making Offers They Can't Refuse | The Godfather influence on all mob media |
| Being a Funny Guy | Goodfellas - Tommy's "Funny how?" scene |
| Visiting the Bing | Sopranos - The Bada Bing strip club headquarters |
| Going to the Mattresses | Mob term for preparing for war |
| Breaking Some Balls | Common wiseguy phrase for teasing |
| Whacking the Competition | Standard mob terminology for elimination |
| Collecting the Vig | Vigorish - interest on loan shark debts |
| Making a Sit-Down | Formal meeting between mob bosses |
| Taking a Trip to Tahiti | Sopranos - Paulie's dream vacation reference |
| Painting Houses | Mob slang for contract killing (blood on walls) |
| Getting Made | Becoming an official member of the family |
| Earning the Button | Another term for becoming a made man |
| Going on a Coffee Run | Sopranos - Christopher's endless coffee runs |
| Handling Waste Management | Sopranos - Tony's "legitimate" business front |
| Meeting with the Gabagool | Sopranos - Tony's favorite capicola cold cut |
| Consulting the Consigliere | The family's trusted advisor role |
| Visiting Junior's Place | Sopranos - Uncle Junior's house/territory |
| Taking the Cannoli | Godfather reference, "Leave the gun, take the cannoli" |
| Leaving the Gun | Godfather reference, complement to above |
| Making Sauce on Sunday | Sopranos - Italian-American family tradition |
| Feeding the Ducks | Sopranos - Tony's beloved pool ducks |
| Talking to Dr. Melfi | Sopranos - Tony's therapy sessions |
| Running the Pork Store | Sopranos - Satriale's Pork Store HQ |
| Shining the Shoes | Goodfellas - Billy Batts' fatal insult to Tommy |
| Getting the Envelopes | Regular tribute payments to the boss |
| Keeping the Mouth Shut | Omerta - the code of silence |
| Showing Some Respect | Essential mob etiquette |
| Playing the Shy | Running a loan sharking operation |
| Clipping Coupons | Mob euphemism for taking out targets |
| Going Down the Shore | Sopranos - Jersey Shore references |
| Making the Collections | Weekly rounds to collect protection money |
| Watching the Rat | Looking out for FBI informants |
| Kissing the Ring | Showing respect to the boss |
| Walking the Walk | Proving loyalty through action |
| Keeping It in the Family | Mob business stays internal |
| Settling Old Scores | Revenge is a dish best served cold |
| Making Things Disappear | Bodies, evidence, problems |
| Running Numbers | Old school illegal lottery operation |
| Counting the Take | Dividing up the earnings |
| Checking the No-Show Job | Sopranos - Union jobs that require no actual work |
