# Mob Classics

Spinner verbs inspired by **Goodfellas** and **The Sopranos** - the greatest mob stories ever told.

## Sample Verbs

- Getting the Papers for 30s
- Handling Waste Management for 30s
- Visiting the Bing for 30s
- Feeding the Ducks for 30s
- Making Offers They Can't Refuse for 30s
- Breaking Some Balls for 30s

## Files

- [mob-classics.json](./mob-classics.json) - Configuration file (42 verbs)
- [mob-classics.md](./mob-classics.md) - Full documentation with references

## Usage

```bash
# Copy to your Claude Code settings
cp mob-classics.json ~/.claude/settings.json
```

Or merge the `spinnerVerbs` section into your existing settings.
