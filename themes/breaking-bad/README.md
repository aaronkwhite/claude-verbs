# Breaking Bad Theme

A Claude Code spinner verb theme inspired by Breaking Bad. Transform your coding sessions with references to Walter White's journey from chemistry teacher to Heisenberg.

## Sample Verbs

- **Saying My Name** - "Say my name."
- **Knocking** - "I am the one who knocks."
- **Applying Science** - "Yeah, science!"
- **Being the Danger** - "I am the danger."
- **Collecting Minerals** - "They're minerals, Marie!"

## Files

- [`breaking-bad.json`](./breaking-bad.json) - Configuration file for Claude Code
- [`breaking-bad.md`](./breaking-bad.md) - Full reference with all 40 verbs and their origins

## Usage

Copy `breaking-bad.json` to your Claude Code configuration directory, or merge the `spinnerVerbs` configuration into your existing settings.

## What It Looks Like

```
Saying My Name for 30s
Cooking for 12s
Calling Saul for 45s
Being the Danger for 8s
```
