# Die Hard Spinner Verbs

Action hero spinner verbs for Claude Code, featuring 40 verbs inspired by John McClane's legendary battles against terrorists, mostly in buildings he did not intend to visit that day.

## Sample Verbs

- Yippee Ki Yay-ing for 30s
- Crawling Through Vents for 30s
- Walking on Broken Glass for 30s
- Ho Ho Ho-ing for 30s
- Welcoming to the Party for 30s

## Files

- [die-hard.json](./die-hard.json) - JSON configuration for Claude Code
- [die-hard.md](./die-hard.md) - Full reference with all verbs and movie sources

## Usage

Copy `die-hard.json` to your Claude Code configuration directory or copy the JSON block from the markdown file directly into your settings.

## Movies Featured

- Die Hard (1988) - The one that started it all at Nakatomi Plaza
- Die Hard 2 (1990) - Dulles International Airport chaos
- Die Hard with a Vengeance (1995) - Simon says in New York City
- Live Free or Die Hard (2007) - Cyber terrorism meets old-school action
- A Good Day to Die Hard (2013) - Father-son bonding in Russia

*Note: This theme focuses primarily on the original film, because let's be honest, that's where the magic lives.*
