# Lord of the Rings Theme for Claude Code

Transform your Claude Code spinner into a journey through Middle-earth with 40 verbs inspired by Peter Jackson's Lord of the Rings trilogy.

## Sample Verbs

- **Carrying the Ring to Mordor** for 30s
- **Not Simply Walking into Mordor** for 30s
- **Lighting the Beacons of Gondor** for 30s
- **Speaking Friend and Entering** for 30s
- **Keeping It Secret, Keeping It Safe** for 30s

## Files

| File | Description |
|------|-------------|
| [lord-of-the-rings.json](./lord-of-the-rings.json) | JSON configuration for Claude Code |
| [lord-of-the-rings.md](./lord-of-the-rings.md) | Full verb list with references and quick copy block |

## Installation

Copy the contents of `lord-of-the-rings.json` to your Claude Code configuration file, or use the Quick Copy JSON block in the markdown documentation.

## Coverage

Verbs are drawn from all three films:
- The Fellowship of the Ring (2001)
- The Two Towers (2002)
- The Return of the King (2003)

Includes iconic moments like the Doors of Durin, Helm's Deep, the Beacons of Gondor, and the climactic journey to Mount Doom.
