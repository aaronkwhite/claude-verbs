# Matthew McConaughey Theme

A laid-back, philosophical Claude Code spinner verb theme inspired by Matthew McConaughey's iconic roles and his distinctively Texan approach to life.

## Sample Verbs

- Alright Alright Alright-ing for 30s
- Time Being a Flat Circle for 30s
- Just Keep Livin-ing for 30s
- Lincoln Commercial-ing for 30s
- Going Through the Wormhole for 30s

## Files

- [matthew-mcconaughey.json](matthew-mcconaughey.json) - JSON configuration for Claude Code
- [matthew-mcconaughey.md](matthew-mcconaughey.md) - Full verb list with references

## Usage

Copy `matthew-mcconaughey.json` to your Claude Code configuration, or use the Quick Copy block in `matthew-mcconaughey.md`.
